## Instructions

### Setup

```shell
git clone https://github.com/Parag-kumbhar/myportfolio.git
cd cleanfolio
```

If you use [npm](https://github.com/nvm-sh/nvm) or [yarn](https://github.com/Schniz/fnm), execute:

```shell
npm install
```



To install and launch the project, run these commands:

```shell
npm
npm start
```

### How to Use

- Open the `public/index.html` file and replace:

`<title>Parag</title>` with `<title>Parag Kumbhar</title>`.

- Open the `src/portfolio.js` file and make the necessary changes.

### Deployment

- In the `package.json` file, update:

`"homepage": "https://github.com/Parag-kumbhar/myportfolio.git"`

to `"homepage": "https://Parag-kumbhar.github.io"`.

- Push the changes to your repository.

- To build and deploy, run the following commands:

```shell
npm build
npm deploy
```
